# この研修で理解できるようになること（5分）

> この研修では、**AI アシスタント「Claude Code」に日本語で指示を出しながら、AI チャットアプリを1日で作ります。**
> プログラミングの経験がなくても大丈夫です。以下のことが理解できるようになります。

---

## AI を使った開発の進め方

- **Claude Code（AI アシスタント）に自然な日本語で指示を出し、アプリを開発する方法** がわかる
- 「どこまで AI に任せられるか」「どう指示すれば思い通りのコードが出るか」の感覚がつかめる
- エラーが出たときに AI に貼り付けて解決してもらうデバッグの流れが身につく

---

## Web アプリの全体像

- **フロントエンド（画面）→ API（サーバー）→ データベース（保存）** という Web アプリの基本構造が理解できる
- 「ユーザーがボタンを押す → API が呼ばれる → DB からデータを取得 → 画面に表示」という一連の流れがわかる

![Web アプリの基本構造](/docs/00-web-app-architecture.png)

---

## 使う技術の基礎知識

研修を通じて、以下の技術がどんなものか・何のために使うのかが理解できるようになります。

| 技術             | 理解できること                                                      |
| ---------------- | ------------------------------------------------------------------- |
| **Next.js**      | React ベースの Web アプリの作り方、ページの構成方法                    |
| **TypeScript**   | 型があるとなぜ安全なのか、JavaScript との違い                         |
| **Hono**         | API（サーバー側の処理）の作り方、リクエストとレスポンスの仕組み        |
| **PostgreSQL**   | データベースにデータを保存・検索する仕組み                            |
| **Drizzle ORM**  | SQL を書かずにデータベースを操作する方法                               |
| **pgvector**     | テキストを数値（ベクトル）に変換して「意味の近さ」で検索する仕組み     |
| **Gemini AI**    | AI に質問して回答を生成させる方法、Embedding（ベクトル変換）の仕組み  |
| **shadcn/ui**    | 既成の UI パーツを組み合わせて画面を作る方法                          |
| **Neon**         | クラウド PostgreSQL サービスを使ってデータベースをセットアップする方法 |

## この研修で学べること

- Claude Code に日本語で指示を出して **アプリを動かすところまで体験** できる
- **Web アプリの全体像**（画面 → API → DB）の流れがつかめる
- AI API を使って **会話機能やベクトル検索を組み込む方法** がわかる
- エラーが出たときに **AI に貼り付けて解決する** デバッグの進め方が身につく
- 「プログラミングは全然わからない」から「**AI に指示を出せば、ある程度のものが作れる**」に変わる

## この研修では学べないこと

- **プログラミング言語の文法**（TypeScript / JavaScript の書き方を体系的に学ぶ研修ではありません）
- **フレームワークの深い仕組み**（Next.js のルーティングやレンダリング戦略などの詳細）
- **インフラ・デプロイ**（本番環境への公開、CI/CD、Docker などは扱いません）
- **セキュリティ・認証の設計**（OAuth、JWT などの認証フローは対象外です）
- **チーム開発のワークフロー**（Git ブランチ戦略、コードレビュー、PR 運用などは扱いません）
- **AI モデルの仕組み**（LLM の学習原理やファインチューニングには触れません）
- **3D アバターの動かし方**（VRM / GLB モデルの表示・リップシンク・表情制御などは事前に実装済みです）

> この研修のゴールは「AI を使えば自分でもアプリが作れる」という **成功体験を得ること** です。
> 各技術を深く学びたくなったら、研修後に個別に掘り下げていきましょう。

# 筆者自己紹介ページ デザインドキュメント

## 概要

研修講師の自己紹介ページを `/about` に独立ページとして作成する。

## 要件

- 対象: 研修講師（筆者）の自己紹介
- 配置: 独立ページ `/about`
- コンテンツ: 基本プロフィール（名前、写真、肩書き、紹介文）+ 経歴・実績
- アバター: 画像ファイル（`public/` に配置）
- デザイン: シンプル・モダン（shadcn/ui Card 活用）
- データ管理: TSX 直書き

## ファイル構成

```
app/about/
  page.tsx          — メインページ (Server Component)
public/
  about-avatar.jpg  — プロフィール画像 (ユーザーが配置)
```

## ページレイアウト

```
┌─────────────────────────────────────┐
│  ← ホームに戻る                       │
├─────────────────────────────────────┤
│                                     │
│   [アバター画像]                      │
│   名前                               │
│   肩書き / 職業                       │
│   簡単な自己紹介文                     │
│                                     │
├─────────────────────────────────────┤
│  経歴・実績                           │
│  ─────────                           │
│  • 2020年 - ○○入社                  │
│  • 2022年 - △△プロジェクト           │
│  • ...                              │
│                                     │
└─────────────────────────────────────┘
```

- 上部: プロフィールカード（画像 + 名前 + 肩書き + 紹介文）
- 下部: 経歴・実績セクション（時系列リスト）
- ヘッダーにホームへの戻るリンク

## 技術的詳細

- **Server Component**: データ取得不要の静的ページ
- **スタイリング**: Tailwind CSS + shadcn/ui Card
- **画像**: Next.js `Image` コンポーネントで最適化
- **ナビゲーション**: `next/link` でホームへの戻るリンク
- **データ**: TSX 内にオブジェクトリテラルで定義
- **レスポンシブ**: `max-w-2xl mx-auto` で中央寄せ

## 導線

- 初期実装では他ページからのリンクは追加しない
- URL `/about` で直接アクセス可能
- 必要に応じて後からリンクを追加
